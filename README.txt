Телеграмм-бот для водителей компании SkyBuild

Бот позволяет водителю управлять аккаунтом на сайте через Телеграмм (использует API сайта)
Основные фичи:
- Разделы "Мои задания" и "Задания на сегодня"
- Возможность выбрать задание из списка и
    1) посмотреть информацию о нем
    2) начать или завершить его
    3) посмотреть контрольные листы
    4) посмотреть задачи в рамках путевого листа
- Возможность выбрать нужный контрольный лист и пройти его (ввод ответов на вопросы)
- Возможность выбрать нужную задачу, посмотреть ее описание и
    1) завершить или продолжить задачу
    2) посмотреть список результатов
    3) добавить, редактировать или удалить результат


Начало работы:
Чтобы запустить бота, необходимо
    1) создать файл .env с BOT_TOKEN
    2) скачать Docker и в Терминале в Pycharm ввести   docker compose up -d
